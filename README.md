# SS Basic Calculator 🧮

A sleek, modern, and fully functional **Scientific Calculator** built with a focus on clean UI and smooth user experience. This project features a "Glassmorphism" design and supports basic arithmetic as well as essential scientific functions.

## ✨ Features

- **Glassmorphism UI:** A modern aesthetic using backdrop-filters and semi-transparent layers.
- **Scientific Operations:** Support for `sin`, `cos`, `tan`, `log10`, `ln`, `square roots`, and constants like `π` and `e`.
- **Memory Functions:** Includes `MC` (Clear), `MR` (Recall), `M+` (Add), and `M-` (Subtract) for complex calculations.
- **History Log:** Tracks your last 10 calculations so you never lose your data.
- **Responsive Design:** Optimized for both desktop and mobile viewing.
- **Error Handling:** Built-in validation to handle syntax errors gracefully.

---

## 🛠️ Tech Stack

- **HTML5:** For structured layout.
- **CSS3:** Custom styling featuring CSS Gradients and Flexbox/Grid.
- **JavaScript (ES6):** Logic handling, string parsing, and mathematical computations.

## 📖 How to Use

1. **Basic Math:** Use the number pad and standard operators (`+`, `-`, `*`, `/`).
2. **Scientific Mode:** Click on functions like `sin` or `log`. 
   - *Note: Trigonometric functions (sin/cos/tan) are calculated in **Degrees**.*
3. **Memory:** Use `M+` to save a result and `MR` to bring it back to the display.
4. **History:** Refer to the right-hand panel to see your previous calculation steps.
